    Movie Classifier

    I built a classifier using a train and test set of data about movies. My k-nearest-neighbors-classifier guesses whether a movie is a comedy or a thriller, using only the number of times words appear in the movies' screenplay.

    Diet and Disease

    The purpose of this project was to conduct A/B tests and bootstrap samples to see if there was a relationship between diet and the risk of getting cardiovascular diseases.

    Gitlet

    Gitlet is a version-control system similar to Git. This system can add, remove, and trackmodified files. I implemented hashmaps and tree data structures that store committed files and their previous versions so that they can be easily accessed for future use.
